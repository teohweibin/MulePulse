# Secrets

This folder is gitignored. Create these 3 files manually before running the project.

## db_password.txt
Any strong password for the PostgreSQL database.

## jwt_secret.txt
Any long random string used to sign JWT tokens.

## openrouter_key.txt
Your OpenRouter API key from https://openrouter.ai/keys

Without the OpenRouter key the API still works — AI reports will fail gracefully.
