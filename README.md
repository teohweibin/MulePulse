# MulePulse

Dependency-free frontend prototype for a graph-native mule network detection and investigation workspace.

Built for **NexHack 2026 Track 2: Fintech Risk & Fraud Intelligence**.

## Run

Open `index.html` directly in a browser.

## Scope

- Public no-login landing page
- Guided case demo for non-technical users
- Mule cluster graph
- Risk queue
- Explainable case file
- Threshold tuning
- Human-in-the-loop action log
- NexHack judging rubric alignment

## Demo Positioning

MulePulse focuses on one practical workflow: helping fraud analysts detect and disrupt coordinated mule networks before victim reports arrive. The product is designed to show:

- Real fintech risk impact
- Graph-native technical execution
- Explainable AI investigation workflow
- Human-controlled enforcement actions
- Commercial adoption path for banks, e-wallets, payment processors, and regtech vendors

See `docs/PRD.md` for product requirements.
